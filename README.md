# Buscador de anagramas

#### Como instalar:
  * Baixe e instale o [XAMPP] ou [WAMP];
  * Baixe o projeto;
  * Caso esteja usando XAMPP:
    * Extraia a pasta do projeto para o diretório HTDOCS dentro da pasta raiz do XAMPP;
  * Caso esteja usando XAMPP:
    *  Extraia a pasta do projeto para o diretório WWW dentro da pasta raiz do WAMP;
  * Acesse http://localhost/NOME_DA_PASTA;
  * Digite a palavra que deseja buscar os anagramas e clique no botão "Buscar anagramas".
  


[XAMPP]: <https://www.apachefriends.org/pt_br/index.html>
[WAMP]: <http://www.wampserver.com/en/>
